# E-Commerce Customer Churn Analysis (SQL)

## Objective
Analyze customer churn patterns using SQL to identify key business insights.

## Tools Used
- MySQL

## Dataset
- Customer dataset including tenure, payment mode, satisfaction score, and order behavior

## Steps Performed
- Data Cleaning (handled missing values, removed outliers)
- Data Transformation (created new columns like ChurnStatus)
- Data Analysis using SQL queries
- Used JOINs, GROUP BY, CASE statements

## Key Insights
- Customers with complaints have higher churn rate
- Certain city tiers show higher churn
- Payment methods influence customer retention

## Business Impact
- Helps identify high-risk customers
- Supports retention strategies
